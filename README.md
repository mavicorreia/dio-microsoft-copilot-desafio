# Desafio Microsoft Copilot Studio 

Através do Copilot Studio pode-se criar uma ferramenta que permite criar assistentes inteligentes (Copilotos) personalizados. Ele utiliza o poder da IA generativa para responder a perguntas, automatizar fluxos de trabalho e se integrar com diversas fontes de dados.

Pode ser criado o nome do agente, a descrição do assistente e instruções de como o Copilot deve ser executado.
